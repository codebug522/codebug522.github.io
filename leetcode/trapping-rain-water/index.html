LeetCode Solution: Trapping Rain Water

üíß Problem: Trapping Rain Water

Description: Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it can trap after raining.

Example:

Input: height = [0,1,0,2,1,0,1,3,2,1,2,1]
Output: 6

üß† Algorithm Explanation

To calculate trapped water at every index i, we need to find:

  • The maximum height on the left of i
  • The maximum height on the right of i
  • Water trapped at i = min(left_max, right_max) - height[i]

To improve efficiency:

  • We precompute two arrays left_max[i] and right_max[i]
  • Traverse once to build them, then another pass to compute trapped water

üêç Python Code

def trap(height):
    if not height:
        return 0

    n = len(height)
    left_max = [0] * n
    right_max = [0] * n

    left_max[0] = height[0]
    for i in range(1, n):
        left_max[i] = max(left_max[i - 1], height[i])

    right_max[n - 1] = height[n - 1]
    for i in range(n - 2, -1, -1):
        right_max[i] = max(right_max[i + 1], height[i])

    total = 0
    for i in range(n):
        total += min(left_max[i], right_max[i]) - height[i]

    return total
