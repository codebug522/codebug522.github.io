LeetCode Solution: Word Ladder

üß© Problem: Word Ladder

Description: Given two words, beginWord and endWord, and a dictionary word list, return the length of the shortest transformation sequence from beginWord to endWord, such that:

  • Only one letter can be changed at a time.
  • Each transformed word must exist in the word list.

Example:

Input: beginWord = "hit", endWord = "cog", wordList = ["hot","dot","dog","lot","log","cog"]
Output: 5  # "hit" -> "hot" -> "dot" -> "dog" -> "cog"

üß† Algorithm Explanation

This is a shortest-path problem in an unweighted graph. Each word is a node, and there is an edge between two words if they differ by one letter. We use Breadth-First Search (BFS) from beginWord to find the shortest path to endWord. To speed up, we precompute generic patterns like h*t, ho* etc. to represent transformations.

üêç Python Code

from collections import deque, defaultdict

def ladderLength(beginWord, endWord, wordList):
    if endWord not in wordList:
        return 0

    wordSet = set(wordList)
    pattern_map = defaultdict(list)

    for word in wordList:
        for i in range(len(word)):
            pattern = word[:i] + '*' + word[i+1:]
            pattern_map[pattern].append(word)

    queue = deque([(beginWord, 1)])
    visited = set([beginWord])

    while queue:
        word, length = queue.popleft()
        for i in range(len(word)):
            pattern = word[:i] + '*' + word[i+1:]
            for neighbor in pattern_map[pattern]:
                if neighbor == endWord:
                    return length + 1
                if neighbor not in visited:
                    visited.add(neighbor)
                    queue.append((neighbor, length + 1))
    return 0

‚úÖ Complexity

  • Time: O(N * M^2) where N is word count and M is word length
  • Space: O(N * M)
