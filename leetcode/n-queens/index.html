LeetCode Solution: N-Queens

üëë Problem: N-Queens

Description: The N-Queens puzzle is the problem of placing n queens on an n x n chessboard such that no two queens attack each other. Return all distinct solutions to the N-Queens puzzle.

Example:

Input: n = 4
Output: [[".Q..","...Q","Q...","..Q."]]

üß† Algorithm Explanation

This is a classic backtracking problem, which we solve using an explicit stack to avoid recursive or nested functions. Each item on the stack holds:

  • The current queen position history (path, a string of column indices)
  • The current row
  • The columns used so far
  • The positive diagonals (row + col)
  • The negative diagonals (row - col)

We keep placing queens row by row and push all valid paths onto the stack. When a path reaches n rows, it’s a valid solution.

üêç Python Code (Single Function Version)

def solveNQueens(n):
    res = []
    stack = [("", 0, set(), set(), set())]  # path, row, cols, diag1, diag2

    while stack:
        path, row, cols, diag1, diag2 = stack.pop()

        if row == n:
            board = []
            for r in range(n):
                line = ["."] * n
                line[int(path[r])] = "Q"
                board.append("".join(line))
            res.append(board)
            continue

        for col in range(n):
            if col in cols or (row + col) in diag1 or (row - col) in diag2:
                continue
            stack.append((
                path + str(col),
                row + 1,
                cols | {col},
                diag1 | {row + col},
                diag2 | {row - col}
            ))

    return res

‚úÖ Complexity

  • Time: O(N!) (exploring valid queen permutations)
  • Space: O(N^2) for result and temporary board construction
